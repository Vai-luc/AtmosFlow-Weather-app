# Blinkit Grocery Data Analysis

## 📊 Project Overview
This is a comprehensive data analysis project on Blinkit grocery store performance and sales trends. The project includes exploratory data analysis (EDA), SQL-based queries, and interactive Power BI visualizations.

## 📊 Analysis Includes

### Data Cleaning
- Standardized Item Fat Content (LF → Low Fat, reg → Regular)
- Handled missing values and inconsistencies

### Key Visualizations
✅ Sales by Fat Content (Pie Chart)  
✅ Sales by Item Type (Bar Chart)  
✅ Sales by Outlet Location (Horizontal Bar)  
✅ Sales by Outlet Size (Pie Chart)  
✅ Sales Trends by Establishment Year (Line Chart)  
✅ Fat Content Distribution by Outlet Tier  

### SQL Analytics
- Total sales and average metrics
- Sales breakdown by item type
- Location-wise performance analysis
- Outlet tier comparisons
- Year-over-year trends

## 🚀 How to Use

### View Dashboard (NO software needed)
1. Click the **Live Dashboard link** above
2. Interact with filters and drill-downs
3. Export reports as needed

### Explore Analysis
1. Open `blinkit_data_analysis.ipynb` in Jupyter Notebook
2. Run cells to see visualizations
3. Modify queries for custom analysis

### Run SQL Queries
1. Import `BlinkIT Grocery Data.csv` to your database
2. Execute queries from `Blinkit_query.sql`
3. Use results for business intelligence

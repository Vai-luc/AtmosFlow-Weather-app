\section{Projects}

\subsection*{AtmosFlow - Professional Weather Dashboard}
\textit{March 2026} | \href{https://github.com/Vai-luc/AtmosFlow-Weather-app}{GitHub Repository} | \href{http://localhost:8000}{Live Demo}

A feature-rich weather application with real-time data integration, interactive maps, and gamification features. Designed with professional UI/UX principles and optimized for performance.

\begin{itemize}[leftmargin=0.15in, itemsep=2pt]
    \item Developed responsive weather dashboard using \textbf{HTML5, CSS3, and vanilla JavaScript ES6+}
    \item Integrated \textbf{OpenWeatherMap API} for real-time weather data and 5-day forecasts across 500+ cities
    \item Implemented \textbf{Leaflet.js} for interactive map background with dynamic location tracking and OpenStreetMap tiles
    \item Created weather-specific particle animations and effects (rain, snow, thunderstorms, clouds, clear skies)
    \item Built gamification system with achievement badges, XP tracking, daily streak counter, and city exploration metrics
    \item Designed professional glassmorphism UI with hardware-accelerated CSS3 animations for smooth 60fps performance
    \item Added favorites system with localStorage persistence and weather trivia game with 6+ educational questions
    \item Integrated dynamic city backgrounds using Unsplash API with curated photos for 20+ major cities worldwide
    \item Implemented responsive design ensuring seamless experience across mobile, tablet, and desktop devices
    \item Optimized code with error handling, performance monitoring, and user-friendly error messages
\end{itemize}

\textbf{Technologies:} HTML5 $\cdot$ CSS3 $\cdot$ JavaScript ES6+ $\cdot$ OpenWeatherMap API $\cdot$ Leaflet.js $\cdot$ Unsplash API $\cdot$ Git/GitHub
